# Space Invaders Java 🚀

Desktop implementation of **Space Invaders** built using **Java Swing**, featuring modular architecture, customizable sprites, selectable bullet types, and event-driven gameplay.

This project demonstrates object-oriented design, UI-driven configuration, and structured package organization typical of academic software engineering projects.

---

## 🎮 Features

- Custom shooter image selection
- Custom invader sprite selection
- Multiple bullet types (fire, ice, earth, custom)
- Background music support
- Menu-driven configuration UI
- Kill counter tracking system
- Event-driven gameplay logic
- Modular architecture using packages
